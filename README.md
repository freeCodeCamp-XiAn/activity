## 欢迎加入 FCC 西安社区贡献开源力量!

### 使命、愿景与口号

长久使命: 技术推进城市发展 —— Code For City

年度使命: 技术解决社区问题 —— Code For Community

社区愿景: 第一个五年, 做西安有价值的互联网技术开源社区

社区口号: 连接与赋能西安开发者

### 社区组织架构

运营团队 contributor: 2~3 人, 负责内容包括技术支持、产品运营、视觉设计和城市推广 

产品技术 tech-leader: 

社区成员: 不限制, 每位成员各显其能，可以在积分平台领取各类任务

### 运营团队 contributor

> P.S.1: 每三个月互相考核一下，如果没有太多贡献，欢迎让位给更多小伙伴~

> P.S.2: 运营团队专注运营任务，负责活动推进、公众号产品体系建设。
